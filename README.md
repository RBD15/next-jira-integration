# React - Jira Integrations

A basic project with React and Jira APIs

## Steps

1. Clone Project
```
  git clone 
```

2. Install packages
```
npm install
```

3. Create .env.local file into the root directory
```
API_URL=https://yoururlproject.atlassian.net/rest/api/latest
JIRA_USERNAME=
JIRA_PASSWORD=
```
